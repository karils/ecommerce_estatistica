# 📊 Dashboard E-commerce com Dash

Este projeto apresenta uma aplicação interativa desenvolvida com **Dash** para análise de dados de e-commerce.

---

## 📈 Funcionalidades

* Histograma
* Gráfico de dispersão
* Mapa de calor
* Gráfico de barras
* Gráfico de pizza
* Gráfico de densidade
* Regressão linear

---

## 🚀 Como executar

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

2. Execute o projeto:

```bash
python app.py
```

3. Acesse no navegador:

http://127.0.0.1:8050/

---

## 📁 Estrutura do projeto

* `app.py` → aplicação Dash
* `ecommerce_estatistica.csv` → base de dados
* `requirements.txt` → bibliotecas necessárias

---

## 🎯 Objetivo

Facilitar a visualização e análise de dados para apoiar a tomada de decisão no e-commerce.
